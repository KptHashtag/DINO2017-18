Opracowanie Git

git config --global user.email ""

Powyższa komenda pozwala na konfiguracje adresu email użytkownika.

git commit -a -m ""

Komenda pozwalająca "commitować" zmiany. Parametr -m pozwala dodać opis rewizji.
Parametr -a pozwala na przeniesienie śledzonych plików, które zostały zedytowane,
do poczekalni.

git diff --staged

Pozwala porównać zmiany między plikami zdalnymi, a plikami przechowywanymi w cache'u.

git log --graph --all

w postaci sformatowanego grafu tekstowego. Np.
